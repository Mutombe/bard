API ENDPOINTS
================================================================================

Base URL: http://localhost:8080/api/v1/

MEDIA ENDPOINTS:
- GET  /media/videos/              - List videos
- GET  /media/videos/{slug}/       - Get video detail
- GET  /media/videos/featured/     - Featured videos
- GET  /media/videos/latest/       - Latest videos
- GET  /media/videos/youtube_search/ - Search YouTube
- GET  /media/videos/youtube_finance/?region={region} - Finance videos
- POST /media/videos/import_youtube/ - Import YouTube video
- GET  /media/categories/          - Video categories
- GET  /media/podcasts/shows/      - Podcast shows
- GET  /media/podcasts/episodes/   - Podcast episodes

NEWS ENDPOINTS:
- GET  /news/articles/             - List articles
- GET  /news/articles/{slug}/      - Article detail
- GET  /news/categories/           - News categories

MARKETS ENDPOINTS:
- GET  /markets/companies/         - List companies
- GET  /markets/indices/           - Market indices
- GET  /markets/tickers/           - Ticker data

AUTHENTICATION:
- POST /auth/login/                - Login
- POST /auth/register/             - Register
- POST /auth/token/refresh/        - Refresh token
- POST /auth/logout/               - Logout
